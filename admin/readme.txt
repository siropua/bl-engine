Локальные секции/модули - те, которые содержатся в каталоге admin САЙТА (ROOT/site/admin/*)
Глобальные - те, которые содержатся в каталоге admin/modules ДВИЖКА (ROOT/bl-engine/admin/modules/*)

Структура конфигов:

Секций:
name = Имя секции
[icon = Иконка секцци (в текущей папке секции)]
Иконка в локальной перезаписывает иконку в глобальной секции
[owerwrite = 1 # если указано в локальной секции - полностью перезаписывает глобальную секцию с таким URL, т.е. от глобальной не остается модулей]
[disabled|hide = 1] отключает/скрывает всю секцию

Модулей:
name = Имя модуля
[icon = Иконка модуля]
[autotabs = 1] Автоматическое подключение файлов табов

Табы создаются в разделе [tab_*] у них только два параметра - name и [icon]
